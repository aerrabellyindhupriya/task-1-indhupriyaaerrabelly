# Emotion-Based AI Chatbot 🤖

## Overview

Emotion-Based AI Chatbot is a Python-based conversational assistant designed to interact with users through simple text conversations. The chatbot can recognize greetings, respond to emotional states, provide motivation, tell jokes, display the current date and time, and engage in basic friendly conversations.

This project was developed as part of the **DecodeLabs Internship Program** to demonstrate the fundamentals of Artificial Intelligence, Natural Language Processing concepts, and chatbot development using Python.

## How It Works

### Greeting Recognition

The chatbot recognizes greetings such as:

```text
hi
hello
hey
hii
whatsup
```

### Emotional Support

The chatbot responds to emotions such as:

* Sad
* Happy
* Stressed
* Lonely

and provides encouraging messages.

### Motivation System

Users can request motivation:

```text
motivate me
motivation
inspire me
```

The chatbot displays a random motivational quote.

### Joke Generator

Users can ask:

```text
tell me a joke
joke
```

The chatbot responds with a random programming or fun joke.

### Date & Time

Commands:

```text
time
date
```

The chatbot displays the current system time and date.

### Exit Commands

Users can end the conversation using:

```text
bye
exit
quit
```
